# Python Quiz Game

A simple console-based quiz game built with Python.

The player answers multiple-choice style questions, receives instant feedback, and gets a final score with a performance rating.

## Features

- Multiple quiz questions
- User input
- Automatic score calculation
- Percentage result
- Performance evaluation
- Case-insensitive answers

## Technologies

- Python 3

## How to Run

1. Clone the repository.

2. Run the program:

```bash
python quiz_game.py
```

## Example

```
===== PYTHON QUIZ GAME =====

What is the capital of France?
Your answer: Paris
Correct!

Which keyword is used to create a function in Python?
Your answer: function
Wrong! The correct answer is: def

===== RESULTS =====

Your score: 4/5
Percentage: 80.0%
Great job!
```
